# PARTSFORM — BRAND SYSTEM (V1.2026)
_For ASI / Autonomous Agents_

---

## 1. BRAND ESSENCE
- Philosophy: **Shibui (Silent Strength)**
- Tone: Industrial, precise, calm authority
- Positioning: **Metal heritage + AI intelligence**
- Never flashy. Never decorative.

---

## 2. LOGO SYSTEM

### Primary Logo
- Forward-tilted, mechanical, wireframe
- Represents motion + infinite supply chain
- Default for headers, website, documents

### Monogram (Turbine)
- Use when space is limited
- App icons, favicons, avatars, loading states
- Represents bearing / AI vision / infinite loop

### Seal (Vertical)
- Institutional authority only
- Contracts, invoices, validation marks
- Simplified “O” in PARTSFORM text

### Logo Rules
- No shadows
- No distortion
- No recoloring
- No busy backgrounds
- Respect clear space at all times

## 4. TYPOGRAPHY

### Primary Typeface — QUANTICO
- Role: Structural authority
- Use for:
  - Headlines
  - Hero numbers
  - SKU highlights
- Rules:
  - ALL CAPS ONLY
  - Never body text

### Secondary Typeface — ANTARTIDA
- Role: Data + readability
- Use for:
  - Body text
  - UI labels
  - Tables, forms, filters
- Rules:
  - Sentence or Title Case
  - Optimized for scanning

### Typography Logic
- QUANTICO = display / impact
- ANTARTIDA = information / flow
- Never mix roles

## 6. VISUAL SYSTEM

- Always visible blueprint grid
- Solid outer frame
- No floating elements

### Containers

- Ruler ticks
- Gold turbine seal = rare premium marker

## 8. ICONS & UI

### Icons
- Standard UI symbols only
- Flat, 2D, no gradients
- Outline → solid on active state

### UI Behavior
- Gold = primary action
- Blue = hover / focus
- Loaders = linear scan (system verifying, not thinking)

- Use active commands
- Eliminate filler words

- Respect silence (white space)
